# Maintenance Inbox Phase 5

Phase 5 should connect property risk controls into the maintenance inbox without changing role boundaries.

## Planned Follow-Up

- Let Eco-Upgrade Planner handoffs create or link a normal work order after landlord approval.
- Show the EPC planning context inside the work order only when the account feature flag is enabled.
- Keep contractors limited to the work-order task, notes and documents intentionally shared with them.
- Let Deposit Vault deductions reference completed maintenance records and invoices from the evidence picker.

## Guardrails

- Do not expose Deposit Settlement Statements to contractors.
- Do not make upgrade recommendations appear as guaranteed outcomes.
- Do not create money movement, rent collection or deposit holding functionality.
- Keep all new surfaces behind account feature flags during rollout.
