# Digital Record Readiness

## Purpose

Digital Record Readiness is a record-keeping signal for MTD preparation. It is not a full MTD compliance score and does not replace accountant review.

## Property Finance Sync Checks

When `mtd_property_finance_sync` is enabled, readiness also looks at Property Finance candidates in the MTD Expense Tracker.

Readiness should not show as complete while:

- synced Property Finance candidates still need review
- possible duplicate candidates are unresolved
- finance-cost review items remain unresolved

The page copy tells landlords: "Property Finance records found. Review them in the MTD Expense Tracker before preparing a quarterly draft."

Reviewed included candidates improve readiness. Excluded candidates are acceptable when the exclusion is documented.
