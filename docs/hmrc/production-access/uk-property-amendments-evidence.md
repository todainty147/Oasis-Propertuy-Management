# UK Property quarterly amendments evidence

An amendment draft can only be created from an original quarterly draft with a successful submission receipt. It preserves the original draft link, receipt/attempt reference, category totals, period, business source, accounting-type snapshot, and an amendment reason.

The amendment is rebuilt from current digital records; arbitrary manual totals are not accepted. It follows the same review, lock, payload validation, duplicate guard, consent-hash, audit, submission, and read-back controls as an original update. The cumulative UK Property `PUT` path is used for tax years from 2025-26.

The accountant pack shows original and amended totals. Fresh explicit consent binds to the amendment draft snapshot before any controlled live pilot. Year-end amendments remain out of scope.
