# API endpoint matrix

| API | E1 use | Mode |
| --- | --- | --- |
| OAuth authorize/token/revoke | Connection lifecycle | Sandbox; controlled future live |
| Business Details v2 | Business source and accounting type | Read |
| Obligations v3 | Quarterly obligations | Read |
| Property Business v6 | UK Property read/read-back | Read |
| UK Property cumulative/period | Original and amendment quarterly update | Sandbox; controlled one-account pilot |

Final declaration, annual submission, foreign property, self-employment, partnership and tax-calculation APIs are not called in E1.
