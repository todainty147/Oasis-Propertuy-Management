# OASIS Marketing Copy Pack

This document is the working copy + SEO content pack for the OASIS marketing site.

Recommended production structure:

```text
oasisrental.com        -> marketing site
oasisrental.com/app    -> current React app
```

Recommended stack for marketing:

- Next.js
- static generation for landing pages and blog
- current React/Supabase app left separate under `/app`

## Brand Positioning

OASIS helps modern landlords manage rentals without juggling spreadsheets, email chains, maintenance chaos, and scattered documents.

Core positioning:

- built for landlords and small property portfolios
- practical day-to-day operations, not enterprise bloat
- maintenance, tenants, finances, and documents in one place
- modern workflow with portals, alerts, and real-time updates

Suggested messaging pillars:

- clarity over chaos
- faster maintenance coordination
- cleaner rental finances
- better tenant communication
- less admin overhead

### Open Graph Defaults

Suggested defaults:

- `og:title`: OASIS Rental
